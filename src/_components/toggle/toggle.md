# Toggle Element



## Toggle with text

```html
<label class="toggle">
  <input type="checkbox" hidden class="toggle-input" />
  <span class="toggle toggle-on">on</span>
  <span class="toggle toggle-off">off</span>
</label>
```

## Toggle with icon

```html
<label class="toggle toggle--icon">
  <input type="checkbox" hidden class="toggle-input" />

  <i class="toggle toggle-on far fa-times-circle"></i>
</label>
```
