## Sizing

### Højder

Auto

```css
height-auto
````

Højde i brøkdele

```css
height-1of1
```

### Viewport height

Vær opmærksom på at disse opfører sig meget forskelligt på forskellige devices

```css
height-100vh
```

### Bredder

Bredde i brøkdele

```css
width-1of1
width-1of2
width-1of3
width-1of4
width-1of5
width-1of6
width-2of3
width-3of4
width-5of6
```

Auto

```css
width-auto
```

### Viewport width

```css
width-100vw
```
